# Validation Assets

This directory contains audio fixtures used by the validation feature.

## validation.wav

A short (3-second) WAV file of a known phrase like "testing one two three"
used by the Validate button to verify the speech engine is working correctly.

**To create this file**: Record yourself saying "testing one two three" and
save as a 16 kHz mono WAV file named `validation.wav` in this directory.
